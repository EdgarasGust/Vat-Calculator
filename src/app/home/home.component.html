Task:

There are a customers and a service providers.

  • The customer can be either a physical person or a legal entity
  • The service provider is a legal entity

Goal:

Allow a service provider to issue an invoice to a client. Customer details, service provider details and the provided services details as well as price and calculated VAT are entered by the user in to the form. When user presses submit button, a dialog window pops out with with the generated invoice and data already filled in.

VAT calculation rules:

  • When the service provider is not a VAT payer - VAT is not calculated on the amount of the order.
  • When the service provider is a VAT payer and the customer:
  • Outside the EU - VAT is 0%
  • Lives in the EU, is not a VAT payer but lives in a different country from the service provider. VAT x% is applied, where x is the percentage of VAT applied in that country, for example: Lithuania 21% VAT
  • Lives in the EU, is a VAT payer but lives in a different country from the service provider. 0% reverse charge applies.
  • When the customer and the service provider live in the same country, VAT always applies

Technical details:

  • Use an API to retrieve countries list (You can find a public one online).
  • Angular
  • SCSS
  • You can use any CSS framework or component library for the UI.
